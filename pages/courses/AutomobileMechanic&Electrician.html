Automobile Mechanic / Automobile Electrician Course

6 Months CBT Level-II, III, IV NAVTTC Certified Multiple Batches

Course Overview

The Automobile Mechanic / Automobile Electrician course is designed to develop the practical skills and technical knowledge required to maintain, service, and repair modern vehicles. This program provides hands-on experience and theoretical understanding of automotive systems, preparing trainees for employment in the automotive industry or self-employment in vehicle maintenance workshops.

CBT LEVEL-2

Course Content

Module 1: Demonstrate Communication Skills

Module 2: Maintain Safe Work Environment

Module 3: Repair Ignition System

Module 4: Repair Fuel System (Carburetor)

Module 5: Repair Engine Cooling System

Module 6: Repair Engine Lubrication System

Module 7: Repair Braking Systems

Module 8: Carry out Wheel Balancing and Alignment

Module 9: Occupational Health and Safety

CBT LEVEL-3

Course Content

Module 1: Perform Service of Electronic Fuel Injection Systems

Module 2: Repair Suspension / Steering System

Module 3: Communicate in the workplace to support customers and team

Module 4: Work effectively in a customer service - sales environment

Module 5: Develop professionalism

Module 6: Comply with health and safety regulations

CBT LEVEL-4

Course Content

Module 1: Perform Overhauling of Engine

Module 2: Repair Manual Transmission System

Module 3: Repair Automatic Transmission System
